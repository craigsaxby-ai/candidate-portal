# Searchline — Candidate Portal

Candidate-facing portal for the Searchline AI recruitment platform.

## Stack
- React + TypeScript + Vite
- Tailwind CSS
- Supabase Auth (candidate accounts)
- Connects to: Searchline Engine API

## Key pages (to build)
- /prescreen/:token — Erica pre-screen interview
- /complete — Post pre-screen completion + profile CTA
- /signup — Candidate account creation
- /login — Returning candidates
- /profile — Candidate profile builder
- /dashboard — My applications

## Dev
npm run dev

## Environment
Copy .env.example to .env and fill in values.
